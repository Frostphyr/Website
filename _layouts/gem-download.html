Installation

Bundler

Add this line to your application's Gemfile:

{%- highlight ruby -%} gem '{{ info.gem }}', '~> {{ page.version }}' {%- endhighlight -%}

NOTE: You may alternatively want to use a version constraint

And then run the command:

{%- highlight console -%} $ bundle install {%- endhighlight -%}

Gem

Run the command:

{%- highlight console -%} $ gem install {{ info.gem }} -v {{ page.version }} {%- endhighlight -%}
